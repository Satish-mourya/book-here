# Doctor Appointment System

A full-stack healthcare management platform built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The application streamlines the appointment booking process by connecting patients with doctors through an intuitive and user-friendly interface.

## Overview

The Doctor Appointment System enables patients to search for doctors, book appointments, and manage their healthcare schedules online. Doctors can manage their availability and appointments, while administrators have complete control over the platform's operations.

## Key Features

* Secure User Authentication and Authorization
* Patient Registration and Profile Management
* Doctor Listing and Detailed Profiles
* Online Appointment Booking and Scheduling
* Appointment Tracking and Management
* Admin Dashboard for System Management
* Responsive Design for Desktop and Mobile Devices
* Real-time Data Management with MongoDB

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Axios
* React Router DOM

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* bcrypt.js
